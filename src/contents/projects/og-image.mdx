這是一個 Next.js 的項目，用於產生具有可自定義參數的 Open Graph (OG) 圖片，例如標題、網址、頭像、作者和日期。使用 [dayjs](https://day.js.org/) 來格式化日期，並使用 `next/navigation` 中的 `useSearchParams` hook 從 URL 查詢字符串中取得參數。產生的 OG 圖像具有固定的寬度和高度，為 1200x630 像素，並具有漸變背景。預設域名為 `honghong.me`，產生的 OG 圖片能顯示作者的頭像、姓名、標題和發佈日期。

## 技術棧

- 框架：[Next.js](https://nextjs.org/)
- 網站圖標：[realfavicongenerator](https://realfavicongenerator.net/)
- 分析：[Vercel](https://vercel.com/)
- 樣式：[Tailwindcss](https://tailwindcss.com)
