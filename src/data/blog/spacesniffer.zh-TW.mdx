## 前言

我們用過 Windows 自帶的磁碟分析工具都知道功能並不強大，今天我就分享一款可視化的磁碟分析工具。

## 安裝

下載網址:

[https://www.fosshub.com/SpaceSniffer.html](https://www.fosshub.com/SpaceSniffer.html)

點擊 Download 即可。

## 使用方法

下載完成後解壓並打開 `SpaceSniffer.exe` 即可。然後選擇你想分析的磁碟，這裏我選擇 C 槽。然後耐心等分析，時間比較久，這取決於你磁碟的大小。
