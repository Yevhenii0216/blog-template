## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/tszhong0411/home.git
```

2. Change the working directory

```bash
cd home
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run dev
```

🌟 You are all set!
