## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/tszhong0411/honghong.me.git
```

2. Change the working directory

```bash
cd honghong.me
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run dev
```

🌟 You are all set!
