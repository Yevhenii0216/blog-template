# 步驟一

我們首先把 `README.md` 檔案放到 Hexo 根目錄中的 `source`。

![](/static/images/blog/hexo-render-2.png)

# 步驟二

去 Hexo 根目錄中的 `_config.yml`，第三十二行左右，把 `skip_render` 新增 `README.md` 即可。

![](/static/images/blog/hexo-render-3.png)

# 最後

---

執行 `Hexo d` 即可。希望能解決你的問題! 😘
