## 安裝 yaml-loader

```bash:Terminal
npm i yaml-loader
```

## 配置 next.config.js

```js:next.config.js
// root/next.config.js
module.exports = {
  webpack: function (config) {
    config.module.rules.push({
      test: /\.ya?ml$/,
      type: 'json',
      use: 'yaml-loader',
    })
    return config
  },
}
```

## 引入 yaml

```js:example.js
// root/pages/example.js
import config from '../config.yaml'

export default function exmaple() {
  return <h1>{config.title}</h1>
}
```

## 備註

如使用 `.tsx` ，請在根目建立 `global.d.ts` ，並輸入:

```ts:global.d.ts
declare module '*.yaml' {
  const data: any
  export default data
}
```

這樣才不會發生錯誤
