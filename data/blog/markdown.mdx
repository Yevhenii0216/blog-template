# Markdown 介紹

---

Markdown 是一種輕量級標記式語言，創始人為約翰·格魯伯。它允許人們使用易讀易寫的純文字格式編寫文件，然後轉換成有效的 XHTML（或者 HTML）文件。這種語言吸收了很多在電子郵件中已有的純文字標記的特性。

# 第一款

---

如果你平常在用 VSCode ，當然可以在延伸模組尋找 Markdown 的編輯器。

名稱: Markdown Preview Enhanced

使用方法: 按一下 `Shift + Ctrl + P` ，上方會彈出一個小視窗，

輸入 HTML:Open Preview to the Side 即可。以後就可以透過快捷鍵 `Ctrl+K` 開啟。

# 第二款

---

名稱: Typora

Typora 也是非常有名，非常好用的 Markdown 編輯器，它的設計理念很不一樣，是真正的即時預覽型編輯器，不同於左右兩個窗口的編輯器。Typora 的設計理念就是極致簡潔，它將「打字」和「預覽」這兩件事情合併了。

使用方法及下載，請到以下官網查看:

[https://www.typora.io/](https://www.typora.io/)

# 第三款

---

名稱: Atom

Atom 也是非常有名的，可以說是專門為程序員推出的一個文本編輯器，界面簡潔，支持實時預覽。功能非常多，除了 Markdown 同時支持 CSS / HTML / JavaScript 等網頁編程語言，還支持自定義，自動分屏功能等。Atom 還具有語義輸入模式，比如輸入 Code 即會自動開啟代碼模式。

使用方法及下載，請到以下官網查看:

[https://atom.io/](https://atom.io/)
