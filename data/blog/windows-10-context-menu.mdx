## 用法

### 變回 Windows 10 樣式

點擊 `變回 Windows 10 右鍵選單樣式.reg` 就可以變回舊版 Windows 10 的右鍵選單樣式

### 恢復 Windows 11 樣式

點擊 `恢復 Windows 11 右鍵選單.reg` 就可以恢復 Windows 11 的右鍵選單樣式
